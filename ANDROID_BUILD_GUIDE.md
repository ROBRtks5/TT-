
# 📱 TITAN: РУКОВОДСТВО ПО СБОРКЕ (ANDROID)

Это пошаговая инструкция по превращению исходного кода в приложение `.apk`.

## 1. Подготовка Окружения

Убедитесь, что у вас установлены:
1.  **Node.js** (v18 или новее).
2.  **Android Studio** (последняя версия).
3.  При установке Android Studio убедитесь, что установлены **Android SDK Tools**.

## 2. Установка Зависимостей

Откройте терминал в папке проекта и выполните команду:

```bash
npm install
```

Затем установите необходимые пакеты Capacitor (если еще не установлены):

```bash
npm install @capacitor/core @capacitor/cli @capacitor/android
npm install @capacitor/app @capacitor/haptics @capacitor/status-bar @capacitor/toast
```

## 3. Инициализация Capacitor

Если папка `android/` еще не существует:

```bash
npx cap init "TITAN Protocol" com.titan.trading.bot --web-dir=dist
npx cap add android
```

## 4. Сборка Веб-Ядра (TITAN OS)

Это скомпилирует TypeScript и React в оптимизированную папку `dist`:

```bash
npm run build
```

> **Важно:** Эту команду нужно выполнять каждый раз после изменения кода, перед синхронизацией с Android.

## 5. Синхронизация с Android

Эта команда копирует папку `dist` внутрь Android-проекта и обновляет плагины:

```bash
npx cap sync
```

## 6. Финальная Сборка (APK)

1.  Откройте проект в Android Studio:
    ```bash
    npx cap open android
    ```
2.  Дождитесь индексации Gradle (внизу справа в Android Studio будет полоса прогресса).
3.  Подключите свой телефон по USB (включите "Отладку по USB" в настройках разработчика).
4.  Нажмите зеленую кнопку **Play (Run)** в верхней панели.

**ИЛИ (Сборка файла без запуска):**
1.  В меню выберите: `Build` -> `Build Bundle(s) / APK(s)` -> `Build APK(s)`.
2.  После завершения появится уведомление "APK(s) generated successfully". Нажмите "Locate".

## 🚀 ПОЗДРАВЛЯЕМ!
Файл `app-debug.apk` — это ваш готовый торговый бот.
Вы можете установить его на любой Android-телефон.

## ⚠️ УСТРАНЕНИЕ НЕПОЛАДОК

**Проблема:** Белый экран при запуске.
**Решение:** Убедитесь, что в `package.json` или `vite.config.ts` параметр `base` установлен правильно (обычно `./` для относительных путей).

**Проблема:** "Capacitor plugin not found".
**Решение:** Выполните `npx cap sync` еще раз.

**Проблема:** "Boot Timeout" в приложении.
**Решение:** У вас медленный телефон. Нажмите "Retry" на экране загрузки. Система самовосстановится.
