# 🛡️ TITAN: BLUE-CHIP VORTEX v2.1 (INTEGRATION PLAN)

## ⚠️ ВАЖНОЕ ПРАВИЛО (LIQUIDITY FUND)
Фонд ликвидности от T-Capital торгуется СТРОГО на СПБ бирже.
- **Тикер:** `TMON@` (обязательно с собачкой на конце).
- **Биржа:** `SPB`.
*Запрещено использовать Московскую биржу для данного актива (внутренний технический запрет брокера).*
