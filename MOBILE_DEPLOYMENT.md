# РУКОВОДСТВО ПО МОБИЛЬНОМУ РАЗВЕРТЫВАНИЮ (TITAN AMM)

Для того чтобы перенести этот проект на ваш Android/iOS терминал, все настройки уже сделаны. Вам нужно лишь следовать этим шагам:

## 1. Сборка Web-версии
Для начала нам нужно скомпилировать React-модуль, чтобы он минифицировался.
```bash
npm run build
```

## 2. Инициализация Capacitor (Уже сделана)
Файл `capacitor.config.ts` уже настроен. `webDir` указывает на папку `dist`.

## 3. Синхронизация платформы Android
Вам нужно добавить модули Android и скопировать туда собранный код:
```bash
npx cap add android
npx cap sync android
```

## 4. Открытие в Android Studio
Откройте проект в Android Studio для финальной сборки APK/AAB:
```bash
npx cap open android
```

В Android Studio:
1. Дождитесь окончания Gradle Sync.
2. Подключите ваш телефон по USB (или выберите эмулятор).
3. Нажмите зеленую кнопку "Run" 🟢 (Shift+F10) для загрузки приложения на устройство.

## 5. Протокол Феникса
Протокол феникса встроен в `src/services/tradeStreamService.ts`. При сворачивании приложения в фон ОС может разорвать вебсокет. При возврате приложения в активный режим, Феникс-протокол автоматически переподключит стримы. Дополнительных вмешательств со стороны нативного кода не требуется.
